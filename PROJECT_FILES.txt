================================================================================
                              API ENDPOINTS
================================================================================

USERS (3 endpoints)
  GET    /api/users                List all users
  GET    /api/users/{id}           Get user profile
  POST   /api/users                Create new user

TWEETS (4 endpoints)
  GET    /api/tweets               List all tweets
  GET    /api/tweets/{id}          Get tweet details
  POST   /api/tweets               Create tweet (auto-hashtags)
  POST   /api/tweets/{id}/like     Like a tweet

HASHTAGS (4 endpoints)
  GET    /api/hashtags             List all hashtags
  GET    /api/hashtags/{id}        Get hashtag + tweets
  GET    /api/hashtags/search/{q}  Search hashtags

SYSTEM (2 endpoints)
  GET    /api/health               Health check
  GET    /                         Frontend UI

================================================================================
                            DATABASE SCHEMA
================================================================================

Users Table
  - id (Primary Key)
  - username (Unique, Indexed)
  - display_name
  - bio
  - location
  - followers_count
  - following_count
  - created_at

Tweets Table
  - id (Primary Key)
  - content
  - author_id (Foreign Key → Users)
  - created_at
  - likes_count

Hashtags Table
  - id (Primary Key)
  - name (Unique, Indexed)
  - created_at

Tweet_Hashtag (Association Table)
  - tweet_id (Foreign Key → Tweets)
  - hashtag_id (Foreign Key → Hashtags)

================================================================================
                           KEY FEATURES
================================================================================

✅ Automatic Hashtag Creation
   - Tweets with #hashtags auto-create hashtags
   - Regex extraction: #(\w+)
   - Idempotent (no duplicates)

✅ Many-to-Many Relationships
   - One tweet → Many hashtags
   - One hashtag → Many tweets

✅ Dynamic Database

================================================================================
                            GETTING STARTED
================================================================================

1. Install Dependencies
   $ pip install -r requirements.txt

2. Seed Sample Data
   $ python3 scripts/seed.py

3. Start Server
   $ python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000

4. Open Browser
   http://localhost:8000

5. Explore API
   http://localhost:8000/docs (Swagger UI)
